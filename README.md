# 💸 ihcux-racha-ai-blazor

Projeto desenvolvido para a atividade prática da disciplina **Interação Humano Computador e UX**, utilizando **Blazor com C#/.NET**.

O objetivo deste projeto é transformar um wireframe do aplicativo **RachaAí** em uma interface funcional, organizada e responsiva, aplicando conceitos de **IHC**, **UX**, **hierarquia visual** e **componentização**.

---

## 📌 Tema do Projeto

### Dashboard de Finanças Coletivas — App **RachaAí**

O **RachaAí** é uma proposta de aplicativo para ajudar usuários a acompanharem gastos compartilhados em grupos, como:

- 🏠 República;
- 🎉 Churrasco entre amigos;
- ✈️ Viagens;
- 🧾 Contas compartilhadas;
- 🍕 Gastos do dia a dia.

A tela desenvolvida representa o **dashboard principal** do usuário, exibindo de forma clara quanto ele tem a receber, quanto precisa pagar e quais grupos estão ativos.

---

## 🎯 Objetivo

Criar uma interface em **Blazor** que represente o painel principal do aplicativo **RachaAí**, colocando em prática os seguintes conceitos:

- 👁️ Hierarquia visual;
- 🧩 Componentização;
- 📱 Responsividade com Bootstrap;
- ✅ Visibilidade do status do sistema;
- 🎨 Uso de cores para indicar situação financeira;
- 🔎 Busca em tempo real;
- 🗂️ Tratamento para lista vazia;
- 🚀 Botão de ação rápida.

---

## 🖼️ Visão Geral da Interface

A tela foi organizada para que o usuário consiga entender sua situação financeira rapidamente.

No topo, são exibidos os principais indicadores:

- 💰 **Saldo Geral**
- 🟢 **Total a Receber**
- 🔴 **Total a Pagar**

Logo abaixo, o usuário encontra uma busca simples para localizar grupos por nome ou categoria. Em seguida, os grupos ativos são exibidos em cards reutilizáveis.
